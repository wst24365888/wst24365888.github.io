找到你要的字體

到Google Fonts 官網尋找你要的字體

下載該字體的安裝包並解壓縮

觀察檔名

這裡拿來示範的是Quicksand這個字體

解壓縮之後

字體檔的副檔名應該是.ttf, .otf或.ttc

而檔名的命名規則是”字體家族名稱-粗細”

ex. Quicksand-Bold.ttf

新建名為fonts的資料夾

並把剛剛解壓縮裡面的字體檔放進fonts內
